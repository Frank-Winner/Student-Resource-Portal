# Student Resource Portal

Version: 1.0

# Project Purpose

The purpose of this project is to create a practical PHP application that combines nearly all PHP fundamentals learned so far into a single manageable project.

The project should be:

✓ Practical

✓ Realistic

✓ Portfolio-worthy

✓ Educational

✓ Not excessively large

✓ Not overly complicated
